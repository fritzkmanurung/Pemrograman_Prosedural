# Pemrograman Prosedural

Repositori ini berisi kumpulan tugas, kuis, dan ujian dari mata kuliah Pemrograman Prosedural.

## Struktur Direktori
- **Exams**: Berisi soal dan jawaban ujian Tengah Semester (Midterm) dan Akhir Semester (Final).
- **Fundamentals**: Berisi materi dasar dan persiapan lingkungan kerja (Setup).
- **Quizzes**: Berisi latihan kuis mingguan.
- **Tasks**: Berisi kumpulan tugas pemrograman prosedural reguler (t01 - t10).

## Teknologi
- Bahasa: C
